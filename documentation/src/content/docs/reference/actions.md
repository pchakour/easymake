This is the list of available action types.

| Name | Description |
| ---- | ---------- |
| [archive](/reference/actions/archive/) | Compress your files as an archive |
| [copy](/reference/actions/copy/) | Copy files or folders to a specific destination |
| [extract](/reference/actions/extract/) | Extract archive |
| [remove](/reference/actions/remove/) | Remove a list of paths |
| [shell](/reference/actions/shell/) | Execute shell command. |
| [git_clone](/reference/actions/git_clone/) | Clone a git repository |
| [move](/reference/actions/move/) | Move files |
